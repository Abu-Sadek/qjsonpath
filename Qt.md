#QtJson Classes

# Introduction #

Qt Json Classes:

|QJsonArray|Encapsulates a JSON array|
|:---------|:------------------------|
|QJsonDocument|Way to read and write JSON documents|
|QJsonObject|Encapsulates a JSON object|
|QJsonValue|Encapsulates a value in JSON|
